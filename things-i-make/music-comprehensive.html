Foreword

I've done a fair bit of music related things since i was a child. As a result of that, i've done things that i'm both proud and not so proud to have done and that's why i don't bring it up much. I'll try to keep the list comprehensive but i'm not going to list out every single recurring weekly recording session or bargig as seperate bullet points. i also started writing this in 2022, so some things i've done far in the past have begun to slip away from my memory.
